# Prompter

A mobile-first teleprompter PWA with live camera preview and recording. Works on iPhone Safari.

## Features

- Paste your script and teleprompter scrolls it over a live camera feed
- Full-screen front camera preview as background
- Semi-transparent white text with dark shadow, readable at arm's length
- Fill light overlay (toggleable, 5-50% brightness) to brighten your face
- Font size controls (16-64px)
- Scroll speed controls (1-10)
- Record with 3-2-1 countdown, auto-downloads MP4 when stopped
- Play/pause scrolling, restart scroll, camera flip
- Wake Lock keeps screen on while filming
- All settings saved to localStorage
- PWA - add to home screen on iPhone
- Dark theme, center eye-line indicator

## Getting Started

```bash
npm install
npm run dev
```

Open on your phone at the local network URL shown in the terminal.

## Build

```bash
npm run build
```

## Deploy

Push to GitHub and connect to Vercel for automatic deployments.
